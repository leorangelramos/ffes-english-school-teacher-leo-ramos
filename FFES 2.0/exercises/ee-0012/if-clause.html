Conditionals (If Clauses)
A Matemática das Consequências

Documentação Estrutural (15 Slides)

1/15 - O Que São If Clauses? ⚖️

As Conditionals expressam que a ação na oração principal (resultado) só acontece se uma certa condição (oração com IF) for satisfeita.

Fórmula: If Clause (Condição) + Main Clause (Resultado).

2/15 - A Regra da Vírgula 🖋️

Se a frase começar com IF, coloque vírgula no meio.
Ex: If it rains, I will stay home.

Se o IF vier no meio, não há vírgula.
Ex: I will stay home if it rains.

3/15 - Zero Conditional (Verdades) 🌍

Uso: Fatos, verdades científicas, coisas que são sempre reais.

Estrutura: If + Present Simple, Present Simple.

Ex: If you heat ice, it melts.

4/15 - First Conditional (Futuro Real) 🔮

Uso: Possibilidades reais e prováveis no futuro.

Estrutura: If + Present Simple, Will + Verbo Base.

Ex: If I study today, I will pass tomorrow.

5/15 - Variações do First Conditional 🛠️

No lugar de 'Will', você também pode usar modais de presente/futuro como Can, May, Should ou até o modo Imperativo no resultado.

Ex: If you see him, tell him to call me. (Imperativo)

6/15 - Second Conditional (Hipotético) 💭

Uso: Situações imaginárias, improváveis ou impossíveis no presente/futuro.

Estrutura: If + Past Simple, Would + Verbo Base.

7/15 - Second Conditional na Prática 🎲

Ex: If I won the lottery, I would buy an island.
(Eu não ganhei e é improvável que ganhe).

8/15 - O Caso do "If I Were" 👑

Na 2nd Conditional, usamos formalmente WERE para todos os sujeitos (I, he, she, it) em vez de 'was'.

Ex: If I were you, I would study. (Se eu fosse você...)

9/15 - Third Conditional (Passado Irreal) 🔙

Uso: Arrependimentos ou críticas. Uma condição no passado que não aconteceu, gerando um resultado também no passado.

10/15 - Estrutura do Third Conditional 🏗️

Estrutura: If + Past Perfect, Would have + Particípio Passado (V3).

Ex: If you had studied, you would have passed.
(Mas você não estudou, então não passou).

11/15 - Variações do Third Conditional ⚙️

No lugar de 'Would have', você pode usar Could have (teria conseguido) ou Might have (talvez tivesse).

Ex: If we had run, we could have caught the train.

12/15 - Unless (A menos que) 🚫

Unless significa "If not". Se você usar 'Unless', a frase fica afirmativa, mas com sentido negativo.

Ex: You will fail unless you study. (= if you don't study).

13/15 - Substitutos do IF 🔗

  • Provided that / As long as: Contanto que.
  • In case: No caso de (precaução).

Ex: I will go provided that you pay.

14/15 - Mixed Conditionals (Avançado) 🔀

Mistura o passado irreal (3rd) com o resultado presente (2nd).

Ex: If I had won the lottery (no passado), I would be rich now (no presente).
