False Friends
Desmascarando as Armadilhas

Documentação Técnica (20 Slides)

1/20 - O que são False Friends? 👯‍♂️

Falsos Cognatos (ou Falsos Amigos) são palavras em inglês que têm grafia ou som muito parecidos com palavras do português, mas significados completamente diferentes.

2/20 - Por que são perigosos? ⚠️

Eles causam mal-entendidos graves. Você pode achar que está dizendo "atualmente", mas está a dizer "na verdade". É a maior fonte de erros nos níveis intermediários.

5/20 - Push vs Pull (A Armadilha da Porta) 🚪

É o clássico erro no shopping:

  • Push: EMPURRAR (Apesar de soar como "puxar").
  • Pull: PUXAR.

7/20 - Library vs Livraria 📚

  • Library: Biblioteca (onde se empresta/lê livros).
  • Bookstore: Livraria (onde se compra livros).

8/20 - Support vs Suportar 🏋️‍♂️

Support: Apoiar, sustentar (uma causa, uma família).
To stand / To bear: Suportar (no sentido de aguentar algo doloroso ou irritante).

9/20 - Lunch vs Lanche 🍔

  • Lunch: Almoço.
  • Snack: Lanche (comidinha rápida).

10/20 - Sensible vs Sensitive 🧠

11/20 - Eventually vs Eventualmente 🔚

Eventually: Finalmente, no fim das contas (após muito tempo/esforço).
Occasionally: Eventualmente (às vezes, de vez em quando).

12/20 - Novel vs Novela 📺

  • Novel: Romance (livro de ficção).
  • Soap Opera: Novela (série de TV).

13/20 - Fabric vs Fábrica 🏭

  • Fabric: Tecido / Pano.
  • Factory: Fábrica.

14/20 - Lecture vs Leitura 📖

  • Lecture: Palestra / Aula expositiva.
  • Reading: Leitura.

15/20 - Resume vs Resumo 📄

  • Resume / Résumé: Currículo (ou o verbo retomar).
  • Summary: Resumo (de um texto, filme).

16/20 - Attend vs Atender ☎️

Attend: Participar, frequentar, estar presente (reunião, aula).
Answer / Serve: Atender (telefone, cliente).

17/20 - College vs Colégio 🏫

  • College: Faculdade / Universidade.
  • High School: Colégio (Ensino Médio).

18/20 - Data vs Data 📅

  • Data: Dados (informações, estatísticas).
  • Date: Data (dia/mês) ou Encontro romântico.

19/20 - Mayor vs Maior 📈

  • Mayor: Prefeito da cidade.
  • Bigger / Larger: Maior (tamanho).

20/20 - Exit vs Êxito 🏆

  • Exit: Saída (porta, escape).
  • Success: Êxito, sucesso.
