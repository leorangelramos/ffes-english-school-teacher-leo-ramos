1/12 - A Lógica do Agrupamento 🧩

Decorar listas alfabéticas é um erro. O cérebro humano aprende por **reconhecimento de padrões**. Ao agrupar verbos que mudam da mesma forma, você cria atalhos mentais permanentes.

2/12 - Verbos Regulares (Padrão Fixo) 🧱

São a maioria esmagadora. A regra é simples: adicione -ED no final para o Passado e para o Particípio.

  • Work - Worked - Worked
  • Play - Played - Played
  • Listen - Listened - Listened

3/12 - Irregulares: Padrão A-A-A (Imutáveis) 🧊

O verbo **não muda nunca**. É a mesma forma no presente, passado e particípio.

BasePastParticiple
CutCutCut
PutPutPut
CostCostCost
HurtHurtHurt

4/12 - Irregulares: Padrão A-B-A (O Retorno) 🔄

O verbo muda no passado, mas **volta à forma original** no particípio.

  • Come - Came - Come
  • Run - Ran - Run
  • Become - Became - Become

5/12 - Irregulares: Padrão I-A-U (O Ciclo) 🔊

Um padrão rítmico onde a vogal muda seguindo a sequência das letras I, A e U.

Base (I)Past (A)Part. (U)
SingSangSung
DrinkDrankDrunk
SwimSwamSwum
BeginBeganBegun

6/12 - Irregulares: Padrão A-B-B (Os Gêmeos) 👯‍♂️

O Passado e o Particípio são **exatamente iguais**. Representam o maior grupo de verbos irregulares.

  • Buy - Bought - Bought
  • Send - Sent - Sent
  • Keep - Kept - Kept
  • Find - Found - Found

7/12 - Irregulares: Padrão A-B-C (A Mudança Total) 🧬

Todas as três formas são diferentes. Geralmente o particípio termina em -N ou -EN.

Base (A)Past (B)Part. (C)
SpeakSpokeSpoken
WriteWroteWritten
SeeSawSeen
TakeTookTaken
GoWentGone

8/12 - O Mistério do "Read" 📖

Parece ser um padrão A-A-A, mas a **pronúncia muda**. No passado e particípio, você fala "RÉD" (como a cor vermelha).

Read (ríd) - Read (réd) - Read (réd).

9/12 - Verbos com Prefixo 🔗

Verbos derivados de outros seguem a lógica do verbo principal.

  • Understand (deriva de stand) -> Understood.
  • Withdraw (deriva de draw) -> Withdrew - Withdrawn.

10/12 - A Matriz de Conjugação (A-B-C) 📋

VerboPresentePassadoParticípio
QuebrarBreakBrokeBroken

CairFallFellFallen
DirigirDriveDroveDriven

11/12 - Resumo Master de Padrões 🏆

Regulares: Terminam em -ED.
A-A-A: Não mudam.
A-B-A: Voltam à base.
I-A-U: Mudança de vogal rítmica.
A-B-B: Passado = Particípio.
A-B-C: Tudo diferente.
