THIRD CONDITIONAL

Matrix do Impossível: If + Past Perfect, WOULD HAVE

XP: 0

📖 Briefing: Arrependimento Hipotético

A Third Conditional fala de algo que não aconteceu no passado.

• NO LADO DO IF: Use HAD + 3ª Coluna.
Ex: If I had gone... (Nunca use Would Have aqui!)

• NO RESULTADO: Use WOULD HAVE + 3ª Coluna.
Ex: ...I would have seen him.

ALERTA DE BUG: O cérebro quer repetir o "would have" no If. Não caia nessa armadilha!
