ZERO CONDITIONAL

Matrix da Lógica: If + Present, Present

XP: 0

📖 Briefing: Verdades Absolutas

A Zero Conditional descreve fatos e regras científicas. A estrutura é sempre PRESENTE em ambos os lados:

• AFIRMAR: If he decides, she accepts. (Cuidado com o "S"!)

• NEGAR: If you don't study, you don't learn.

• PERGUNTAR: Do you cry if he leaves?

ALERTA: Nunca use "Will" aqui. A Zero Conditional é sobre o que sempre acontece.
