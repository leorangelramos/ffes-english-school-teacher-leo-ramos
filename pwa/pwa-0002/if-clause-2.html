SECOND CONDITIONAL

Matrix do Improvável: If + Past, WOULD

XP: 0

📖 Briefing: O Sonho Hipotético

A Second Conditional descreve situações imaginárias ou pouco prováveis no presente.

• A REGRA DO BE: Dentro do If, use WERE para todo mundo (I, You, He, She, We, They).
Ex: If I were rich...

• O PROIBIDO: Nunca coloque "Would" logo após o If.
Ex: If I went... (Certo) / If I would go... (ERRO!)

• O RESULTADO: Use WOULD na outra frase.
Ex: If he accepted, I would celebrate.
